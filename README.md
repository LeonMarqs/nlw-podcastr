## Getting Started

* Clone the project
  ```sh
  git clone https://github.com/LeonMarqs/nlw-podcastr.git
  ```

* After that, install the dependencies listed in `package.json` file.
  ```sh
  npm install 
  #or
  yarn install
  ```

### Run Application

* Development server
  ```sh
  npm run dev
  # or 
  yarn dev	
  ```

* API server
  ```sh
  npm run server
  # or 
  yarn server	
  ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
